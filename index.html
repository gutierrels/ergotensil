El Desafío Diario

Millones de personas enfrentan dificultades diarias al usar utensilios tradicionales para comer. Desde enfermedades degenerativas como Parkinson o esclerosis múltiple, hasta adultos mayores con motricidad fina deteriorada o lesiones temporales.

Esta situación no solo genera frustración y dependencia, sino que también puede afectar negativamente la autoestima y la calidad de vida, convirtiendo un acto tan fundamental como comer en una fuente de estrés.

Ergotensil: La Solución

Hemos desarrollado un set de cubiertos ergonómicos que combinan funcionalidad, diseño y personalización para facilitar el acto de comer de forma autónoma, segura y cómoda.

Nuestro diseño modular, fabricado mediante impresión 3D local, permite una adaptación precisa a las necesidades de cada usuario, promoviendo la independencia y mejorando la experiencia en cada comida. No son solo cubiertos, son una herramienta para la dignidad.

Principales innovaciones:

  • Botón de giro integrado para ajustar el ángulo del cubierto.
  • Pesos equilibrados en el mango para mayor estabilidad.
  • Diseño moderno y elegante, no estigmatizante.
  • Producción local y personalizable mediante impresión 3D.
